# High Ambitions Studio

Single-page premium studio website built with Next.js, React, TypeScript, Tailwind CSS, and Framer Motion.

## Local development

From `C:\Users\balin\High Ambitions` run:

```powershell
npm.cmd install
npm.cmd run dev
```

Then open:

```text
http://localhost:3000
```

## Production build

```powershell
npm.cmd run build
npm.cmd run start
```

## Project structure

- `app/` app router entry and global styles
- `components/layout/` navbar and footer
- `components/sections/` page sections
- `components/ui/` reusable UI building blocks
- `lib/` content and animation helpers
